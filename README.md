# ⭐ 주식 추천 도우미 (초보용)

주식 처음 시작하는 사람을 위해 **지금 들어갈 만한 종목을 점수로 매겨 추천**해 주는 도구.
한국+미국 우량주·ETF를 객관적 지표로 분석하고, 쉬운 말로 이유를 설명합니다.
Next.js(App Router) + yahoo-finance2 로 만들어졌고 **Python 없이** 동작합니다.

> ⚠️ 미래 가격은 누구도 못 맞힙니다. 이 점수는 과거·현재 데이터 기반의 **참고용 ‘분위기 점수’**이며,
> 투자 권유가 아닙니다. 최종 판단과 책임은 본인에게 있습니다.

## 실행 방법

```bash
npm run dev
```

→ 브라우저에서 http://localhost:3000 접속

## 페이지 구성

| 탭 | 내용 |
|---|---|
| ⭐ 오늘의 추천 (`/`) | 후보 종목을 0~100점으로 점수화해 순위로 추천 + 초보 시작 가이드(코어75/새틀25, **변동성 조정 매수금액**) + 종목별 **점수 신뢰도 배지·점수 알파·펀더멘털·원화 수익 분해·1·5년 가격 위치** |
| 📊 종목 분석 (`/analysis`) | 종목별 이동평균·RSI·MACD·거래량 신호 + 차트 (수정종가 기반) |
| ⚖️ 위험 비교 (`/statistics`) | 후보들의 변동성·샤프(통화별 무위험수익률)·MDD 비교표 + 상관관계 히트맵 + **몰빵(집중도) 경고** |
| 🎮 시뮬레이터 (`/simulator`) | **적립식(DCA) 백테스트**(배당 재투자 반영) + **세후 실질수익** + **예적금 대비 초과수익** + **하락장·회복기간** + **위험성향 매칭** |

## 상용 서비스와 다른 점 (특화 지표)

- **🇰🇷 세후 실질수익** — 양도세(미국 22%, 250만 공제)·배당세(15.4%)·환전 스프레드·수수료를 다 뺀 *진짜 손에 쥐는 돈*. 상용 어디에도 없는 한국 투자자 전용 관점. (`src/lib/costs.ts`)
- **📈 점수 알파(정직 지표)** — 점수대로 적립 강도를 조절한 게 *그냥 무지성 적립(DCA)보다 나았는지*를 백테스트로 비교. 못 이기면 "그냥 적립하세요"라고 솔직히 말해요. (`src/lib/alpha.ts`)
- **🧪 점수 신뢰도 배지(백테스트)** — *지난 5년간 70점↑였을 때 이후 6개월 실제 승률·평균수익*과 평소 대비 우위(%p). 생존편향·표본수 경고 포함. (`src/lib/backtest.ts`)
- **⏳ 하락장 회복기간** — 최대낙폭뿐 아니라 *전고점 회복까지 걸린 최장 기간*. 멘탈이 무너지는 건 깊이보다 기간. (`src/lib/recovery.ts`)
- **🏦 예적금 대비 초과수익** — 같은 돈을 정기적금에 넣었을 때와 비교한 진짜 기회비용. (`src/lib/deposit.ts`)
- **💱 원화 실질수익 분해** — 미국주식 수익을 *주가가 번 것 + 환율이 번 것*으로 쪼개요. (`src/lib/fx.ts`)
- **📏 변동성 조정 매수금액** — 출렁이는 종목엔 적게 들어가도록 역변동성으로 금액 배분. (`src/lib/sizing.ts`)
- **🚨 몰빵 경고** — 후보들이 사실상 한 덩어리처럼 움직이면 "분산이 안 됐다"고 경고. (`stats.avgPairwiseCorrelation`)
- **📋 펀더멘털 한 줄 평가 + 5년 밸류 위치** — PER·PBR·ROE·배당과 *5년 가격 백분위*로 "싸냐/비싸냐". (`src/lib/fundamentals.ts`)

> 모든 수익률·지표는 **수정종가(adjclose)** 로 계산해 배당 재투자가 반영됩니다(`src/lib/stocks.ts`).

## 추천 점수는 어떻게 계산되나 (4가지 종합, 100점 만점)

- **추세 (30점)**: 현재가가 이동평균선(20/60/120일) 위에 있어 상승 흐름인가 (정배열)
- **모멘텀 (20점)**: 오르는 힘(MACD)이 있고, 너무 과열(RSI 70↑)되진 않았나
- **안정성 (25점)**: 변동성이 낮고(덜 출렁), 최대낙폭이 얕은가
- **가격 위치/밸류 (25점)**: 1년·5년 가격대에서 무리하게 고점을 추격하는 자리는 아닌가 (싼 자리일수록 가점)

> 추세·모멘텀에 점수가 쏠리면 "이미 오른 종목"을 고득점시키는 고점추격 편향이 생겨,
> 두 항목을 합쳐 10점 줄이고 그만큼 "싼 자리인가(가격위치/밸류)"에 실었습니다.
> 점수는 타이밍 예측이 아니라 **적립 강도 조절**용 참고치입니다.

점수 70↑ = 분위기 좋음 · 50~69 = 무난 · 50 미만 = 관망/약세.
점수 로직은 `src/lib/scoring.ts`(코어 엔진) 에서 조정할 수 있습니다.

## 후보 종목 바꾸기

`src/lib/watchlist.ts` 의 `DEFAULT_WATCHLIST` 를 고치거나, 화면 맨 아래 **관심 종목 추가**로 넣으면
`data/watchlist.json` 에 저장됩니다.

- 미국 주식: 티커 그대로 → `AAPL`, `MSFT`, `VOO`
- 한국 코스피: 종목코드 + `.KS` → 삼성전자 `005930.KS`
- 한국 코스닥: 종목코드 + `.KQ` → 에코프로비엠 `247540.KQ`

```
src/lib/indicators.ts       기술지표 (SMA·EMA·RSI·MACD)
src/lib/stats.ts            통계 (수익률·변동성·샤프·MDD·상관관계)
src/lib/scoring.ts          ★ 점수 코어 엔진 (오늘의 추천 + 백테스트 공용)
src/lib/recommend.ts        추천 빌더 (스코어링·신뢰도·알파·펀더멘털 조립)
src/lib/backtest.ts         점수 신뢰도(성적표) 백테스트
src/lib/alpha.ts            점수 알파 — 점수 적립 vs 무지성 적립 비교
src/lib/costs.ts            한국 투자자 세금·수수료·환전 모델 (세후 실질수익)
src/lib/deposit.ts          예적금 대비 초과수익 계산
src/lib/recovery.ts         하락장 낙폭·회복기간 프로파일
src/lib/simulate.ts         적립식 백테스트 (배당 재투자 반영)
src/lib/fx.ts               원화 실질수익 분해 (주가 + 환율)
src/lib/sizing.ts           역변동성 매수금액 배분
src/lib/fundamentals.ts     펀더멘털 한 줄 해석
src/app/api/recommendations 추천 순위 API
src/app/api/watchlist       후보 CRUD API
src/app/api/analysis/[symbol]  종목별 기술분석 API
src/app/api/compare         후보 위험 비교 API
src/app/api/simulate        시뮬레이터 API (세후·예금비교 포함)
src/app/page.tsx            오늘의 추천 화면
src/app/analysis/page.tsx   종목 분석 화면
src/app/statistics/page.tsx 위험 비교 화면
src/app/simulator/page.tsx  시뮬레이터 화면
```
